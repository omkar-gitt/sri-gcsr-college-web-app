# 🎓 Sri GCSR College Website (Web Project)

## 📌 Overview  
This project is a responsive college website developed for **Sri GCSR College, Rajam**.  
It showcases academic details, courses, campus facilities, blog content, and contact information in a clean and structured interface.

> 🏫 Sri GCSR College  
> Affiliated to Dr. B.R. Ambedkar University, Etcherla  
> GMR Nagar, Rajam, Vizianagaram – 532127  
> Accredited by NAAC with B++ Grade  

---

## 🚀 Features  

- 🏠 Home page with hero section and college introduction  
- 📖 About page with institutional overview  
- 🎓 Courses page (Intermediate & Degree programs)  
- 🧾 Blog page (certifications, internships & programs)  
- 📞 Contact page with location, email & form  
- 🏫 Campus and facilities showcase  
- 🗣️ Student testimonials section  
- 🎨 Modern UI using CSS  

---

## 🛠️ Technologies Used  

- HTML5  
- CSS3  
- Font Awesome

## 💡 Page Descriptions  

- **Home (web.html):** Introduction, courses overview, campus, testimonials  
- **About (about.html):** College vision and overview  
- **Courses (course.html):** Programs offered (Intermediate & Degree)  
- **Blog (blog.html):** Certifications, internships & academic programs  
- **Contact (contact.html):** Location map, email, phone & contact form
